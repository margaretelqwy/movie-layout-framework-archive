# movie-layout-framework-archive
该项目围绕在线电影与电视剧网站的页面布局框架展开整理，对首页模块排列顺序、导航层级逻辑以及播放页信息区块安排进行系统归档。通过结构化文字记录建立资料库，用于分析不同影视平台在页面结构与内容展示方面的变化趋势。仓库仅保存公开页面资料，仅供信息整理与研究参考使用。
